# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

A single-file Java program (`MyTest.java`) that prints `Hello World`. There is no build system (no Maven/Gradle), no test framework, and no dependencies — it is compiled and run directly with the JDK.

## Commands

Run from this directory (`Project 21_ClaudeCodeMasterClass_Jun2026`):

```powershell
javac MyTest.java   # compile -> produces MyTest.class
java MyTest         # run -> prints "Hello World"
javac -version      # confirm a JDK is on PATH
```

`MyTest.class` is a build artifact and can be regenerated by `javac`.

## Conventions

- The `public` class name must match the file name. `MyTest.java` declares `public class MyTest`, so any rename must change both together.
- `public static void main(String[] args)` is the program entry point used by `java MyTest`.
